    Quick Start

    ```elixir def deps do [{:quark, "~> 1.0.0"}] end ``` # What's Included? - A series of classic combinators (SKI, BCKW, and fixed-points), along with friendlier aliases - Fully-curried and partially applied functions - Macros for defining curried and partially applied functions - Composition helpers - Also a composition operator: `<|>` - A plethora of common functional programming primitives, including: - `id` - `flip` - `const` - `pred` - `succ` - `fix` - `self_apply` # Examples

    `defcurry` and `defcurryp`

    ```elixir defmodule Foo do use Quark.Curry defcurry div(a, b), do: a / b defcurryp minus(a, b), do: a - b end # Regular div(10, 2) # => 5 # Curried div.(10).(5) # => 2 # Partially applied div_ten = div.(10) div_ten.(2) # => 5 ```

    `defpartial` and `defpartialp`

    ```elixir defmodule Foo do use Quark.Partial defpartial one(), do: 1 defpartial minus(a, b), do: a - b - c defpartialp plus(a, b), do: a + b + c end # Normal zero-arity one # => 1 # Normal n-arity minus(4, 2, 1) # => 1 # Partially-applied first two arguments minus(100, 5).(10) # => 85 # Partially-applied first argument minus(100).(10).(50) # => 40 # Fully-curried minus.(10).(2).(1) # => 7 ```
